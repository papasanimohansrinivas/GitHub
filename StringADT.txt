####Problem Description
Write a program, to implement ( abstract data type ) String. Following are the operations:
   indexOf(char[]) : Returns the index of parameter char[] in given char[] 
   contains() : Returns a boolean indicating whether the given char[] contains the parameter char[] or not
   endsWith() : Returns a boolean indicating whether the given char[] ends with the parameter char[] or not

# Methods:
   Declare a character array globally.
   Create a class StringADT and a constructor StringADT(char[]) which takes a char[] as parameter and initialize the global character Array.
   1) indexOf(char[]) returns int
   2) contains(char[]) returns boolean
   3) endsWith(char[]) return boolean
